Features of HTML5

Input tag kinds have been expanded

Input is an old attribute that has been resurrected in HTML with new values such as email, month, number, range, search, tel, week, URL, time and etc.
These are the new values that the input tag may include.

ContentEditable

It’s a characteristic that allows us to make changes to the content. It makes determining Where we see Is What we Get a breeze. By clicking on the content, we will be able to change it.

Progress

This tag is used to monitor the status of a job while it is being completed. we may use the progress tag in conjunction with JavaScript. It’s similar to a progress bar.

section

The section tag is used to split a document into sections. For instance, an article may include many parts, such as a header, footer, newest news, and a section for the primary material, among others.

Main

Main is a tag that is used to encapsulate the page’s primary content. There can’t be more than one main tag in a document, and this tag can’t be within the article aside, footer, or header tags. The navigation bar, header, and footer are not included.
